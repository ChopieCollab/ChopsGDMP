# Chop's Godot Multiplayer Framework
A multiplayer project template for future usage.

This project is NOT meant to replace full functionality but rather as a starting off point for prototyping and possibly building multiplayer games.
Technically, you can use this project for singleplayer games as well and simply code the game in a way that would be easier to expand into multiplayer.
